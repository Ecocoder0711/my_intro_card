# My Intro Card - Flutter

A stylish, animated, and personal introduction card built with Flutter. This project showcases a clean user interface with dynamic animations to present a professional and engaging digital card.

<img src="assets/images/My_Intro.png" alt="App Screenshot" width="350"/>

## üìú Description

This Flutter application serves as a digital business card or a personal introduction page. It's designed to be visually appealing, featuring a profile picture, name, professional title, and a personal quote. The app leverages animations to create a memorable user experience, with elements appearing in a staggered sequence and a typewriter effect for the quote.

## ‚ú® Features

- **Clean & Modern UI:** A simple, card-based design that is easy to read and visually pleasing.
- **Staggered Animations:** UI elements (profile picture, name, title, quote) fade in sequentially for a dynamic entry effect.
- **Typewriter Text Effect:** The personal quote is animated with a typewriter effect using the `animated_text_kit` package.
- **Custom Fonts:** Utilizes the `google_fonts` package to enhance typography.
- **Responsive Layout:** The card is designed to be centered and well-proportioned on different screen sizes.
